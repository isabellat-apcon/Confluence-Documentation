# Purpose
This repo provides a rough idea of how to organize documentation for multiple products in one place. The goal is to provide an alternative to Confluence.

Some features about this repo:
1. Users cannot have certain access to folders. Once a user is invited to a repo, they will have access to all it's contents.
2. Every saved change becomes a commit. There is no PR required.
