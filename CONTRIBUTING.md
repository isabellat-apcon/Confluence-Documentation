This file serves as a documentation that outlines rules, standards, and process for developers. 

When creating a new pr:
1. create/edit branch
2. open CONTRIBUTING.md
3. Click one of the pr template links
4. Github will open the pr page
5. Complete PR

Links below opens dif types of template. 
- [Open a general pull request](https://github.com/isabellat-apcon/Confluence-Documentation/compare/main...HEAD?quick_pull=1&template=basic.md)
- [Open a bug-fix pull request](https://github.com/isabellat-apcon/Confluence-Documentation/compare/main...HEAD?quick_pull=1&template=bugfix.md)
- [Open a feature pull request](https://github.com/isabellat-apcon/Confluence-Documentation/compare/main...HEAD?quick_pull=1&template=feature.md)
