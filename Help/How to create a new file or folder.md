## 1. Start from the repo home page

Click **Add file**.

Then choose **Create new file**.

## 2. Create a new folder or file

At the top of the page, type the name of the folder or file you want to create.

### To create a new folder

Type the folder name, then add `/`.

Example:

```text
Product A/
```

GitHub does not save empty folders, so you also need to create a file inside the folder.

Example:

```text
Product A/README.md
```

### To create a new file

Type the file name and end it with `.md`.

Example:

```text
Product A Home.md
```

Markdown files use `.md` at the end.

## 3. Save your changes

Click **Commit changes**.

A box will appear.

The description is optional.

To save the change directly, choose:

```text
Commit directly to the main branch
```

Then click **Commit changes**.

<img width="1911" height="1031" alt="commit changes" src="https://github.com/user-attachments/assets/51be42bc-4ce5-41ad-abae-a85b3d9b8654" />

---

## 4. Confirm the file or folder was created

After committing, go back to the repo home page.

You should see your new file or folder listed.
